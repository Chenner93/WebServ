/var/www/api

🚀 Endpoints Disponibles

GET /api/users

Récupère la liste complète de tous les utilisateurs enregistrés dans le système. 

curl -X GET http://127.0.0.1:8081/api/users \
  -H "Accept: application/json"
🔓 Public ⚡ Rapide 📄 JSON
GET /api/users/{id}

Récupère les informations détaillées d'un utilisateur spécifique via son identifiant unique. 

curl -X GET http://127.0.0.1:8081/api/users/123 \
  -H "Accept: application/json"
🔓 Public 🎯 Paramètre requis
POST /api/users

Crée un nouvel utilisateur dans la base de données avec les informations fournies. 

curl -X POST http://127.0.0.1:8081/api/users \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jean Dupont",
    "email": "jean@example.com",
    "role": "user"
  }'
📝 JSON requis ✨ Création 🔐 Validation
DELETE /api/users/{id}

Supprime définitivement un utilisateur du système en utilisant son identifiant. 

curl -X DELETE http://127.0.0.1:8081/api/users/123 \
  -H "Accept: application/json"
⚠️ Suppression 🎯 ID requis
GET /api/health

Vérifie l'état de santé et la disponibilité de l'API. Retourne le statut du serveur. 

curl -X GET http://127.0.0.1:8081/api/health
💚 Health Check 🔓 Public
POST /api/data

Envoie des données volumineuses pour traitement. Supporte jusqu'à 50MB de données. 

curl -X POST http://127.0.0.1:8081/api/data \
  -H "Content-Type: application/json" \
  -d '{"data": "..."}'
📦 50MB Max ⚙️ Processing
